# CuroVia

**CuroVia** is a full-stack telemedicine platform designed to connect patients with verified healthcare professionals through secure video consultations. By combining role-based access control, real-time appointment scheduling, virtual healthcare services, and subscription-based monetization, CuroVia delivers a modern, scalable, and production-ready healthcare SaaS experience.

---

## Key Features & Functionalities

### Authentication & Role-Based Access
- Secure authentication powered by Clerk.
- Join the platform as either a Patient or Doctor.
- Personalized dashboards based on user roles.
- Protected routes and role-based permissions.
- Admin-controlled onboarding and verification workflow.

### Doctor Discovery & Specialty Search
- Browse verified healthcare professionals.
- Search doctors by specialty.
- View doctor profiles and availability.
- Find the right healthcare provider for specific medical needs.

### Real-Time Appointment Booking
- Book consultations based on doctor availability.
- Real-time slot management and scheduling.
- Easy appointment booking workflow.
- Cancel or manage appointments directly from the dashboard.

### HD Video Consultations
- One-on-one virtual consultations.
- Join appointments directly from the platform.
- Camera and microphone controls.
- Secure and seamless telehealth experience.
- Conduct consultations from anywhere in the world.

### Subscription & Credits System
- Credit-based consultation booking system.
- Monthly subscription plans.
- Purchase additional credits when needed.
- Track available credits from the dashboard.

### Doctor Dashboard
Doctors can:
- Manage consultation availability.
- View upcoming appointments.
- Join patient consultations.
- Track earnings and consultation history.
- Request withdrawals via PayPal.
- Manage professional activity from a centralized dashboard.

### Admin Dashboard & Verification System
Admins can:
- Review doctor applications.
- Approve or reject healthcare professionals.
- Suspend or manage doctors.
- Process withdrawal requests.
- Monitor platform activity and healthcare operations.

### Responsive Healthcare Experience
- Fully responsive across desktop, tablet, and mobile devices.
- Modern healthcare-focused user experience.
- Fast and intuitive navigation.
- Beautiful UI built with Shadcn UI and Tailwind CSS.
